Minecraft - Telegram bridge, build on top of mineflayer &amp; telegraf.

## Features

- Listen mode
- Chest interact
- Inventory support

## Roadmap / Upcoming Features

- Auto reconnect
- Multi user support
- Simple navigation
- Pm / Whisper notification
